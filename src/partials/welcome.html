Welcome to Web Init!

Web Init is a modern starter kit for developing websites which includes a preconfigured build process and a project structure that can be built upon.

This package comes preinstalled with:

  • SASS
  • PostCSS
  • Autoprefixer
  • ESLint
  • Airbnb Style Guide
  • JQuery
  • Webpack 4

Thank you for using Web Init. Happy coding!
